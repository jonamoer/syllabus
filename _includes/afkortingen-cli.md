*[chmod]:   change mode
*[ls]:      list files and directories
*[mkdir]:   make directory
*[rm]:      remove
*[rmdir]:   remove empty directory

*[.]:       current directory
*[..]:      parent directory

*[g]:       global
*[i]:       install
*[up]:      update

*[-g]:      --global
*[-v]:      --version
*[-la]:     long list + list all files and directories
*[-rf]:     recursive + force
